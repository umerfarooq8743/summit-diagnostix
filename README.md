# Summit Diagnostix Website

## Overview
The Summit Diagnostix website is designed to provide information about innovative specimen collection systems for clinical and veterinary industries. The site features a clean, clinical theme with a focus on usability and responsiveness across devices.

## Project Structure
```
summit-diagnostix
├── index.html
├── about.html
├── products.html
├── quality.html
├── blog.html
├── orders.html
├── contact.html
├── styles.css
├── scripts.js
└── README.md
```

## Features
- **Home Page**: Includes a hero section with the logo and tagline, an introductory paragraph, and buttons for exploring products and contacting the company.
- **About Us**: Provides information about the company's mission and values, along with a section highlighting the advantages of Summit Diagnostix consumables.
- **Products**: Outlines various product categories and details the advantages of Urine Collection Systems, along with future development platforms.
- **Quality & Support**: Emphasizes the company's commitment to quality and invites collaboration with partners.
- **Blog**: Showcases the latest articles related to specimen collection workflows and diagnostic consumables.
- **Orders**: Contains an order form for users to submit inquiries about products.
- **Contact**: Provides contact information and a team section.

## Setup Instructions
1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the website.
3. Ensure that all files are in the same directory for proper linking of styles and scripts.

## Technologies Used
- HTML5
- CSS3
- JavaScript (vanilla)

## Responsive Design
The website is designed to be responsive, ensuring a seamless experience on desktop, tablet, and mobile devices.
